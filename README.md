# SCENT ID

A premium interactive fragrance discovery prototype built as a frontend-only luxury MVP concept for L'Oréal Luxe.

## App structure

- `index.html`
  - Loads the React runtime, Tailwind CDN configuration, fonts, and mounts the app.
- `script.js`
  - Contains the component-based React application, navigation flow, reusable UI building blocks, and quiz/result logic.
- `data.js`
  - Stores all mock data arrays and objects for fragrances, moods, reviews, videos, articles, discovery kits, and user dashboard content.
- `styles.css`
  - Adds premium visual refinements such as gradients, scrollbar polish, floating motion, and shimmer loading effects.

## Experience map

The prototype includes:

1. Landing page
2. Multi-step onboarding / SCENT quiz
3. SCENT ID result page
4. Fragrance detail page
5. Discover by mood page
6. Learn perfumery page
7. Lifestyle match page
8. Discovery kit page
9. Refill & sustainability page
10. Profile / dashboard page
11. Community page

## Notes

- Frontend only; no backend required.
- Uses mock JSON-like data in separate constants.
- Optimized for responsive presentation-style screenshots and startup MVP demos.
